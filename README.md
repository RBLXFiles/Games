# RBLXFiles - Roblox Files

Welcome to RBLXFiles. This is an open space focused on archiving, preserving, and providing historical Roblox files and assets for developers and enthusiasts.

##  Roblox Game Files
This project is a collection of games that have been uncopylocked. The repository associated with this project is used to preserve games that are considered at risk to become unavailable in the future or that are already unavailable. 

These games include:
* Games that have previously been uncopylocked but are not anymore.
* Official ROBLOX games (because they are notorious for having been disabled, copylocked again, or removed in the past).
* Games that are reasonably likely to become unavailable at some point.

## Guidelines and Contributions
Anyone may suggest the addition of a game to this collection. This can be done on GitHub by sending a pull request or filing an issue on the repository associated with this collection. This repository is owned by the RobloxLabs organization.

Please note the following restrictions:
* This collection does not include games that have been stolen.
* Games of a low quality or with almost no content should not be submitted.
* One may legitimately use material from any of these games in any project.
